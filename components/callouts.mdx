# Callouts

Use callouts to draw attention to important information without breaking the reading flow.

## Example

<Callout kind="tip">
  Use environment variables to store sensitive configuration like API keys.
</Callout>

<Callout kind="info">
  Keep callouts short so they stay easy to scan.
</Callout>

<Callout kind="alert">
  Double-check destructive actions before proceeding.
</Callout>
