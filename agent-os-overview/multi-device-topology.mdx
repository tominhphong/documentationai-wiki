Hệ thống Agent OS của Phong chạy trên 3 thiết bị vật lý, mỗi máy một vai trò chuyên biệt. Chúng nối với nhau qua Tailscale mesh network và LAN nội bộ, tạo thành một cơ sở hạ tầng phân tán mà Claude Desktop + agents có thể điều hành từ bất kỳ nơi nào. Trang này liệt kê từng thiết bị, topology mạng, và phân chia trách nhiệm.

## Device Inventory

Agent OS hoạt động theo nguyên lý "mỗi máy, một vai trò" — không có máy chủ trung tâm, mà mỗi thiết bị chuyên môn hóa để làm việc hiệu quả nhất cho loại task của nó. Dưới đây là bảng tóm tắt:

| Thiết bị | Hostname (Tailscale) | Spec | Vai trò chính | Tailscale IP |
|----------|----------------------|------|---------------|--------------|
| Mac Studio | `phongs-mac-studio` | 36GB RAM / 512GB SSD | Brain + editing — Opus orchestrator, Claude Code CLI chính | `100.98.44.123` |
| Mac mini M4 | `phongrealtortxs-mac-mini` | 16GB RAM / 256GB SSD | VPS 24/7 — agents scheduled/long-running, AI agent gateway | `100.98.146.85` |
| UM790 | `minisforum` | Windows, 32GB RAM / 1TB SSD | Chạy phần mềm Win-only (Hidemium anti-detect, Boxphone farm) | `100.71.136.88` |

### Mac Studio — Trung tâm điều phối

Đây là bộ não chính của Agent OS. Phong ngồi trực tiếp tại Mac Studio để điều hành công việc hàng ngày.

**Vai trò:**
- Chạy Claude Desktop ở chế độ Cowork Max 20x — Opus 4.7 làm orchestrator cho toàn bộ hệ thống
- Chạy Claude Code CLI, ClaudeKit plugins, và các tools cần latency thấp (web browser, code editor, file manager)
- Làm nơi Phong thao tác trực tiếp khi cần kiểm soát tay hoặc quyết định nhanh
- Quản lý research, planning, và strategic thinking

**Software stack:**
- Claude Desktop + Cowork mode
- Claude Code CLI với shell sandbox
- ClaudeKit Engineer + Marketing plugins
- Homebrew toolchain (Git, Node.js, Python, Docker CLI)
- Tailscale client

**Ghi chú:**
- Spec 36GB RAM cho phép chạy song song nhiều Claude sessions + browser tabs + IDE
- System hostname: `PhongsMacStudio`. Tailscale hostname: `phongs-mac-studio`. Tailscale IP: `100.98.44.123`
- Đã join tailnet ngày 2026-04-24

### Mac mini M4 — VPS 24/7

Mac mini là "worker 24/7" không cần tương tác. Nó chạy các agent lâu dài, scheduled tasks, và đóng vai trò AI agent gateway cho các máy khác.

**Vai trò:**
- Host GoClaw (Glow + ClaudeKit Workflow wrapper) — API gateway để các agents từ Mac Studio hoặc UM790 gọi long-running tasks
- Chạy cron jobs và scheduled workflows (ví dụ: daily report generation, batch processing, monitoring)
- Xử lý các tasks không cần interactivity (background inference, file sync, log aggregation)
- Làm backup executor nếu Mac Studio offline

**Software stack:**
- Docker + docker-compose
- Tailscale client (kết nối vào mesh)
- GoClaw service (bao gồm Claude API client, task queue)
- Git client (pull private repos)
- Time-sync tools (cron, systemd timers)

**Ghi chú:**
- Spec 16GB RAM đủ cho 1–2 concurrent Sonnet-4.6 instances + GoClaw daemon
- Tailscale hostname: `phongrealtortxs-mac-mini`. Tailscale IP: `100.98.146.85`
- System hostname: `phongrertxsMini` (resolved qua mDNS `phongrertxsMini.lan`). LAN IP: `192.168.8.107`

### UM790 — Windows execution

UM790 là máy chạy Windows dành cho phần mềm chỉ tồn tại trên nền Windows — không cần hoặc không nên virtualize.

**Vai trò:**
- Chạy Hidemium anti-detect browser (cloud botting, multi-account automation)
- Chạy Boxphone farm — giả lập điện thoại ảo để test app, automation, hoặc cloud phone services
- Dự phòng: nếu cần Windows-only tools tương lai (VBA macros, proprietary software, etc.)

**Spec cao:**
- 32GB RAM (chạy song song nhiều Boxphone instances + Hidemium profiles)
- 1TB SSD (lưu trữ VM images, browser profiles)

**Software stack:**
- Windows 11 Pro
- Hidemium browser
- Boxphone manager
- Tailscale client (kết nối vào mesh)
- Optional: Synergy/KVM client để chia sẻ keyboard/mouse từ Mac Studio

**Ghi chú:**
- Tailscale hostname: `minisforum` (đặt theo brand Minisforum của UM790). Tailscale IP: `100.71.136.88`
- System hostname: `Minisforum` (resolved qua mDNS `Minisforum.lan`). LAN IP: `192.168.8.174`
- UM790 không chạy Claude Desktop — không cần. Thay vào đó, Mac mini/Mac Studio gửi task via Tailscale RPC

<Callout kind="info">
Tất cả hostname (system + Tailscale) và IP (LAN + Tailscale) đã được xác nhận ngày 2026-04-24. System hostnames resolved qua mDNS Bonjour reverse-lookup (`dns-sd -Q`).
</Callout>
